Pour jouer au jeu il faut utliser le click gauche pour faire bouger le joueur dans la scene, E pour intéragir avec les PNJ, et O pour utiliser les items.
Au début du jeu vous commencer avec 1 item infinis qui est un poisson. Votre but est de récupéré assez d'argent pour acheté des items plus puissant et battre les ennemis qui bloque le chemin vers la sortie.

J'ai beaucoup de difficulté sur le code, j'ai surtout eu du mal à le comprendre mais j'ai réussis à faire un jeu où le joueur peut se blesser et blesser ses ennemis.
